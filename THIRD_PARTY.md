# Third-Party Materials

This document tracks third-party materials that are included in this repository
or used to generate artifacts committed to it.

> Rule of thumb: this project can only license what it has rights to license.
> Third-party materials keep their original license/terms.

- See `pyproject.toml` / `poetry.lock` (or equivalent) for dependency licensing.

## AI-generated content

If you commit AI-generated images/text, document:

- Tool/model used (e.g., "Stable Diffusion XL", "DALL·E", etc.)
- Prompt (optional, but helps provenance)
- Any applicable terms of service / usage restrictions (if relevant)
- Whether the output was edited, and by whom

Example:

- `docs/images/example.png`
  - Generated with: TOOL/MODEL
  - Date: YYYY-MM-DD
  - Terms: (link or short note)
  - Notes: (e.g., edited in GIMP, overlays, etc.)
